Variables BDFD soportadas

Puedes usar variables de Bot Designer For Discord en textos y URLs:

  • $username - Nombre del usuario
  • $nickname - Apodo en el servidor
  • $serverName - Nombre del servidor
  • $memberCount - Conteo de miembros
  • $authorAvatar - Avatar del usuario

Consejos

  • Usa fondos con degradado para un aspecto más profesional
  • Combina efectos con opacidad baja para mejores resultados
  • Usa texto con sombra para mejorar la legibilidad
  • No sobrecargues la tarjeta con demasiados elementos
